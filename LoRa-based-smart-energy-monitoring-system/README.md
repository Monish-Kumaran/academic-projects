# 🔋 LoRa-Based Smart Energy Monitoring System

## 📖 Overview
This project implements a **smart energy meter** using LoRa (Long Range) communication.  
It allows **real-time monitoring** of household electricity consumption (voltage, current, and power) with reliable long-distance data transmission.

---

## 🛠️ Tools & Technologies
- **Hardware:** Arduino UNO, LoRa Ra-02 Module, ACS712 Current Sensor, ZMPT101B Voltage Sensor, OLED Display
- **Software:** Arduino IDE, LoRa.h library, Adafruit GFX & SSD1306 libraries
- **Communication:** LoRa (433/868 MHz)

---

## ⚙️ Features
- Real-time monitoring of energy consumption  
- Long-range communication (2–15 km)  
- OLED display for local feedback  
- Scalability for **smart grid integration**

---

## ▶️ How to Run
1. Connect sensors to Arduino UNO as per the circuit diagram.  
2. Upload the transmitter and receiver codes using Arduino IDE.  
3. Power the modules and observe real-time values on the OLED display.  

---

## 📌 Applications
- Smart grids  
- Remote energy monitoring  
- Automated billing systems
